Remote Agents 是为现代开发者与 AI 协同设计的远程 Web 终端与文件管理网关。
完美集成 Zero-Latency 终端 (ttyd+tmux)、全功能文件浏览器、自动 HTTPS 证书及按需公网穿透隧道。

快速安装体验 核心功能解析
scott@workbench: ~/remote-agents

强悍内核，为高效而生

融合极致的 Web 终端性能与企业级安全网关，告别繁琐配置，开启真正的移动编码时代。

⚡ 零延迟 Web 终端

基于高性能 xterm.js 与 WebSocket 技术，内置 tmux 深度融合。网络断线、浏览器刷新，终端运行会话瞬时无缝重连，永不断线。

📂 全能文件管理器

左侧树形结构 + 核心平铺视图。支持一键重命名、极速下载、全功能文本语法高亮编辑；新近加入 HTML 与 PDF 文件的高清 16:9 全屏预览。

🔒 零配置权威绿锁

自动生成 10 年期自签名 ECDSA P-256 HTTPS 证书。同时能深度绑定 Tailscale 官方证书体系，跨设备访问自动匹配安全权威绿锁。

🌐 按需公网穿透隧道

一行参数 `-tunnel` 开启 Cloudflare 隧道。Go 守护程序自动按需缓存引擎。智能分配全球互联网公网 URL 与扫码登录二维码，免去 NAT/端口映射配置。

🤖 CC-Connect AI 消息桥接

打通飞书、Telegram、Slack、Discord。将你的项目工作区动态注册为智能体；甚至可以在飞书中发一句“开启工作台”，智能体便会在服务器后台直接一键拉起公网。

🎙️ 原生 Speech-to-Text

集成 Web Speech 原生语音识别机制，支持中英文快捷听写，享受真正零等待的“口谕式”声控智能体开发体验。

⚡ 极速响应，永不断线

底层依赖高性能的 C 语言 `ttyd` 守护进程，通过高速 WebSocket 通道将终端输出瞬间投射在浏览器的 `xterm.js` 画布上。结合 `tmux` 会话自动挂载技术，即便你在地下铁或电梯里信号中断，连回后依然能够瞬间恢复现场，确保代码在后台安全不中断地编译运行。

  • 毫秒级字符渲染延迟
  • tmux 会话自动恢复
  • 移动端 Quick Keys 虚拟键盘
  • 单击 tmux 滚动模式便捷切换

🔒 绿锁就绪，保护隐私

语音麦克风录入、Service Worker 离线机制、系统剪贴板等现代高级 Web API 强制要求 HTTPS 安全通道。Remote Agents 在 `--ssl` 指令启动时，若未检测到可用证书，将智能生成 10 年期高强度的自签名 ECDSA P-256 证书。此外，如果你配合 Tailscale 运行，它能自动发掘并加载 Tailscale Let's Encrypt 证书，全球各地无感秒开绿锁 🔒。

  • 自动 ECDSA P-256 自签名证书
  • Tailscale 证书全自动扫描与匹配
  • 强制全流量 SSL/TLS 加密传输
  • 安全剪贴板与麦克风授权

🌐 内网穿透，一键发布

无需公网 IP、不需要任何路由器配置或烦人的 DDNS。通过加装 `-tunnel` 选项，Go 后端守护程序在检测到本地无 cloudflared 时，会显示百分比进度并自动从 GitHub 下载（大小仅 30M，海外秒开，国内 15 秒极速缓装，二次启动 0.1 秒瞬开）。它会自动获取公网专属 URL，并在终端显示高对比度二维码，手机一扫，公网秒连！

  • 一行参数 `-tunnel` 启动
  • 智能 PATH 复用与一次性高速缓装
  • 终端控制台高对比度扫码连线
  • 动态随机安全 Token 安全锁
scott@macbook $ remote-agents -tunnel
[tunnel] cloudflared not found. Starting automatic download... [tunnel] Download finished in 2.4s. Starting cloudflared tunnel... [tunnel] Workspace published securely to public web: 👉 https://remote-agents-scott.trycloudflare.com (QR rendered in console)

清晰分层，极简分发

高性能的 C 后端引擎与极富扩展性的 Go 守护程序深度融合，实现一次打包，极简部署。

Frontend (html/)

使用 Preact + TypeScript 开发。经过编译打包及 Gzip 压缩后，以嵌入式 C 字节数组 `html.h` 的形式直接融入终端服务器，实现无静态文件依赖分发。

Go Agent (agent/)

系统的管理中枢。控制多工作区切换、生成自签名/Tailscale SSL 证书、对接 Cloudflare 公网安全隧道以及反向代理 ttyd 终端与文件操作 API。

Terminal Server (src/)

基于高性能 C 语言 `ttyd` 构建的底层 WebSocket-to-TTY 桥接器。深度挂接本地 bash、zsh 和 `tmux` 维持进程永不断线，实现极致的毫秒级字符映射。

仅需几秒，立即体验

我们提供了多样化的安装手段。推荐使用最简便的 NPM 全局安装或者使用 root Makefile 编译打包。

npm install -g @scottzx/remote-agents

全局包包装器会自动根据系统类型从高速镜像拉取匹配的预编译二进制执行包。

  • 系统要求：Node.js >= 22 (完美兼容最新的 Node 24 运行环境)
  • 一键启动：remote-agents -tunnel
npx @scottzx/remote-agents -tunnel -ssl

完全免安装，利用 NPX 工具按需高速拉取并在终端运行，即停即走。

  • 在局域网下用平板等访问时，建议加上 -ssl 参数获取 HTTPS 麦克风和剪贴板特权。
docker run -d -p 8080:8080 -v /path/to/workspaces:/workspace --name remote-agents scottzx/remote-agents:latest

基于官方 Docker 镜像秒级挂载你的开发区运行，方便云服务器部署。

# 1. 编译 ttyd 后端 git clone --recursive https://github.com/scottzx/remote-agents.git cd remote-agents && mkdir build && cd build cmake .. && make # 2. 构建前端 (Yarn 3) cd ../html yarn install && yarn build # 3. 编译 Go 守护进程 cd ../agent go build -o remote-agents ./cmd/agent/main.go

项目在根目录提供了一个超级高效的 Makefile 来简化所有步骤，克隆后在根目录下直接执行一键编译：

  • make all：编译所有终端、前端、Go 引擎与 cc-connect 守护模块。
  • make package：一键打包出带有宿主主机名、系统以及架构的完整 `.tar.gz` 部署包。
